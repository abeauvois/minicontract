# Setup

- forked from

```
git clone https://github.com/abeauvois/minicontract.git
```

- Navigate to the directory

```
cd minicontract
```

- Install the dependencies with

```
pnpm
```

- Configure the .env file with your firebase credentials
- run with

```
pnpm dev
```
